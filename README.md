
# Todo

![alt text](https://i.imgur.com/UeuN7cA.jpg)

Simple ToDo app built on Angular 10 framework.

## What you can find here?

- CRUD operations
- IndexedDB handling
- Validations
- Modals
- Drag and drop
- Toast notifications
- Dark mode styling

## Launching of project

Run `npm start` then navigate to `http://localhost:4200/`.

## Technologies

 - Angular 10
